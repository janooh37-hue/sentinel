💻 Install the App Certificate — Windows PC

GSSG Manager · one-time setup (about 1 minute)

To open the app at https://gssgit on a Windows PC, install this small security certificate once. It simply tells Windows to trust our office server. Safe — it holds no passwords.

First: copy the file gssg-ca.crt onto the PC (USB drive, email, or a shared folder).

Install steps

  1. Double-click the file gssg-ca.crt.
  2. In the window that opens, click Install Certificate…
  3. For Store Location, choose Local Machine, then click Next.
    (Approve the admin / User Account Control prompt if it appears.)
  4. Choose "Place all certificates in the following store", then click Browse…
  5. Select "Trusted Root Certification Authorities" → click OK → Next → Finish.
  6. A security warning appears — click Yes to confirm.
  7. You'll see "The import was successful." Click OK.
  8. Open Edge or Chrome and go to https://gssgit. ✓ The app loads with a padlock and no warning.
"Local Machine" is greyed out / no admin rights? Choose Current User instead in step 3 — the rest of the steps are identical, and it works for your login.

Which address to use

  1. https://gssgit — simplest, works on Windows.
  2. https://gssgit.local — also works (and is what phones use).

❓ If it doesn't open

  1. Make sure the PC is on the same office network as the server.
  2. Type the full address with https: https://gssgit
  3. If you see a certificate warning, the cert wasn't placed in "Trusted Root Certification Authorities" — repeat steps 4–5.
